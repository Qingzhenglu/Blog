## Electron技术架构

Electron = Chromium + Node.js + Native apis

Electron集成了Chromium与Node.js，是一个实现了跨平台的桌面应用开发框架。
Electron通过使用JavaScript、HTML 和 CSS等Web技术构建桌面应用程序。

- Chromium: 支持最新特性的浏览器
- Node.js: javascript 运行时，可实现文件读写等
- Native APIs：提供统一的原生界面能力
